# ⬡ PharmaCosmos
### *Drug Intelligence Atlas*

> An immersive 3D-styled pharmaceutical reference website covering 55+ commonly used drugs — built for pharmacy students and healthcare enthusiasts.

---

## ✨ Features

- 🌌 **Deep space UI** with animated star field & orbital rings
- 💊 **55+ drugs** across 8 pharmacological categories
- 🔍 **Live search** — filter by name, class, or indication
- 📂 **Category filter** — Analgesics, Antibiotics, Cardio, CNS & more
- 🧪 **Rich drug cards** — structure, properties, uses, dosage forms & posology
- 📱 **Responsive** — works on mobile & desktop
- 🔗 **Link** — https://demonxyonko.github.io/Pharma/
